# 0x03 RT-AK + STM32Cube AI

- 硬件平台： `ART-PI`

- 模型：`model/mnist.tflite`

- 我想要从零开始到 `Mnist_RTT` 工程？

  1. 先打开 `RT-Thread Studio` 新建 `ART-PI` 模板示例工程
  2. 下载 `RT-AK` 
  3. 运行 `RT-AK`
  4. 即可

  > `Mnist_RTT/applications` 中的 `mnist_app.c` 为示例开发 `Demo`，并非 `RT-AK` 生成

  > 该问题是另外一个话题了，详细请查阅 RT-AK 相关文档。
